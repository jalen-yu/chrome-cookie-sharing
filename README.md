## :dart: About

针对 Chrome 浏览器的 Cookie 共享插件，解决本地开发 Cookie 携带问题。

## :checkered_flag: Starting

```bash
# Clone this project
$ git clone https://github.com/jalen-yu/chrome-cookie-sharing.git

# Access
$ cd chrome-cookie-sharing

# Install dependencies
$ pnpm install

# Run the project
$ npm run build

```
